⏱️ Workout Timer
Un'applicazione web minimalista e reattiva per gestire i tempi di recupero e le sessioni di allenamento, progettata per essere rapida, intuitiva e senza distrazioni.  

🚀 Progetto Lampo
Sono entusiasta di condividere questo progetto!  

La cosa incredibile è che l'intera applicazione è stata sviluppata e resa funzionante in meno di 10 minuti utilizzando Google AI Studio.  

Questo esperimento dimostra quanto sia diventato potente il binomio tra creatività umana e intelligenza artificiale nel prototipare strumenti utili in tempi record.  

🛠️ Tech Stack
React + Vite (per una velocità di build fulminea)  

CSS3 (per un'interfaccia pulita e mobile-friendly)  

Google AI Studio (come co-pilota per lo sviluppo rapido)  

⚡ Caratteristiche
Interfaccia pulita e ad alto contrasto.  

Timer preciso per sessioni HIIT, Tabata o recuperi classici.  

Completamente responsive (perfetto per lo smartphone in palestra).  

📦 Installazione e Avvio Locale
Clona la repository:

git clone [url-tua-repo]

Installa le dipendenze:

npm install
    ```
Avvia il server di sviluppo:
    ```bash
    npm run dev
    ```
